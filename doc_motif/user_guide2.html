Writing Mail

The compose window is shown when one of the compose menu items is selected. The window allows you to fill out the header lines (as specified in your .privrc), select signing & encryption etc. options and write your message (of course).

Multiple compose windows can be open at any time.

When selecting one of the Reply menu items which specifies "(include)" will cause the currently displayed message to be inserted in the message indented with the indent string specified in your .privrc (or "> " otherwise).

Including files Arbitrary files can be included by clicking on the Insert... button. This will show a filer which enables you to select a file to insert into the message.
Including messages Any message can be included into the message area of a compose window. To achieve this simply select the message you want included from the message list (which becomes a primary selection) and paste it into the compose_message area by clicking the middle button (the default action for pasting a primary selection).

It is also possible to drag the message from the message list and drop it into the compose_message area.

Folders

Selecting Folders from the View menu will cause a folder browser to be displayed. This allows a hierarchical structure of mail folders to be browsed and a current folder to be specified for loading, copying or moving mail into.

In addition to the Folder Browser, the combobox in the toolbar contains the list of commonly accessed mail folders as specified in the .privrc file.

When a folder is clicked on in the Folder Browser, the folder name is inserted into the combobox and that folder becomes the current folder. Also any name can be typed in the combobox to allow messages to be saved to an arbitrary file.

Loading folders When there is a current folder specified, selecting Load folder from the Folder menu will cause the current folder to be loaded, replacing the currently displayed messages with the contents of that folder.

An alternate method is to double click on the folder icon in the Folder Browser.

Saving to a folder Selecting Copy to folder or Move to folder from the Folder menu will cause the currently selected message(s) to be saved to the current folder. If move was selected then the original messages will be deleted as well.
